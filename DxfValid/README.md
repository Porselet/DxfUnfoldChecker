# DxfUnfoldChecker 🛠️📐

**DxfUnfoldChecker** — это специализированный инженерный валидатор плоских DXF-разверток листового металла, сгенерированных в CAD-системах. Приложение разработано на современном стеке **.NET 10.0** и **WPF** для автоматического контроля геометрии перед отправкой деталей на станки лазерной, плазменной или гидроабразивной резки с ЧПУ.

В отличие от встроенных проверок в популярных CAD (например, Компас-3D), данный инструмент не просто ищет зазоры, а проводит глубокую топологическую дефектоскопию чертежа, выявляя скрытые дубликаты линий гиба, наложенные друг на друга контуры и специфические артефакты экспорта.

---

## 🚀 Основные возможности

*   **Пакетный анализатор «Светофор»**: Проверка не отдельных файлов, а сразу всего комплекта разверток в папке за один клик. Система мгновенно выставляет статусы:
    *   🟢 **ГОДЕН** — идеальная геометрия, готовая к резке.
    *   🟡 **ВНИМАНИЕ** — контуры замкнуты, но обнаружены технологические особенности (выпрямление дуг, скрытые блоки).
    *   🔴 **БРАК** — критические дефекты (разрывы контура или наложения линий).
*   **Двухступенчатый контроль качества**:
    1.  **Проверка замкнутости (Топология)**: Сборка отрезков «за кончики» с настраиваемым инженерным допуском (по умолчанию `0.05 мм`). Позволяет найти микро-зазоры, из-за которых деталь не выпадет из листа после реза.
    2.  **Инспектор чистоты (Наложения)**: Поиск коллинеарных линий, перекрытий «внахлест» и абсолютных дубликатов, которые заставляют лазер проходить по одному месту дважды, вызывая порчу металла.
*   **Умная обработка артефактов Siemens NX**: Автоматическое выявление и корректная обработка «вырожденных» дуг гигантского радиуса (вплоть до 8 000 км) и микро-углов, которые обычно ломают сетку рендеринга графических движков.
*   **Графический движок в стиле AutoCAD**:
    *   Плавный интерактивный Zoom колесиком мыши строго в точку курсора (без скачков координат).
    *   Панорамирование (Pan) чертежа зажатием левой кнопки мыши.
    *   Цветовая маркировка: чистый контур — серый, наложения — бирюзовый (`Cyan`), разомкнутые участки — красный.
*   **Визуальные маркеры-мишени**: Подсветка точных мест разрывов крупными контрастными фиолетово-желтыми точками, которые легко обнаружить на общем плане и детально рассмотреть при приближении. Возможность скрывать маркеры одной галочкой на панели.
*   **Автоматические отчеты**: Генерация подробного технического файла `Валидация_Комплекта_Отчет.txt` в целевой папке со статистикой для конструкторов.

## 🛠️ Технологический стек и архитектура

Программа спроектирована по принципам **SOLID** и разделена на независимые, легко тестируемые классы-сервисы (архитектурный паттерн "Фасад"):
1.  **`DxfExtractor`** — извлекает сырую геометрию из основного пространства и внутренних блоков (`Inserts`), линеаризует окружности/дуги в плоские сегменты `AnalyticSegment`.
2.  **`TopologyEngine`** — строит граф связей и формирует замкнутые/разомкнутые цепочки геометрии.
3.  **`CleanlinessChecker`** — проводит векторный анализ отрезков на параллельность и пересечение интервалов проекций.
4.  **`DetailViewerWindow` (WPF)** — отвечает за интерактивный рендеринг и интерфейс взаимодействия с инженером-технологом.

**Используемые библиотеки:**
*   Платформа: `.NET 10.0 (WPF)`
*   CAD-парсер: `netDxf` (с поддержкой извлечения примитивов напрямую из структуры документа).

## 📦 Как запустить проект

1.  Убедитесь, что у вас установлена **Visual Studio 2025/2026** с поддержкой **.NET 10.0**.
2.  Склонируйте репозиторий:
    ```bash
    git clone https://github.com
    ```
3.  Откройте файл решения `DxfUnfoldChecker.sln`.
4.  Восстановите NuGet-пакеты (пакет `netDxf` подтянется автоматически).
5.  Нажмите **F5** для сборки и запуска приложения.
